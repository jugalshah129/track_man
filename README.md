# Track - Man

App for organizing track day event.

## what is this?

This web app allows different user profiles to manage and execute track day event. 

## who this is for

- Rides Planner
- Event Managers
- OEMs

## how it works

Use loin as admin to set up profiles. Then create events and track execution modules.  
| Function | What it does |
|---------|-------------|
| `Receptiont` | Register participant for an event |
| `Trainer` | Marked trained participant |
| `Gear Manager` | Updated which riding gears are given and what are received |
| `Track Manager` | Marks entry and exit from the track |
